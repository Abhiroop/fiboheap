# fiboheap

Purely functional fibonacci heaps

| Operation     | Complexity    |
| ------------- |:-------------:|
| empty         | O(1)          | 
| minimum       | O(1)          |
| insert        | O(1).Amortized|
| union         | O(1).Amortized|
| extractMin    | O(log n).Amortized|
